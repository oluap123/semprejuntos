Criar uma pasta para o projeto e rodar no cmd:

# Iniciar o composer:
    composer init 

# Preencher os dados solicitados 

# Alterar o arquivo composer para :
         "psr-4": {
            "App\\": "App/"

# Iniciar o servidor do PHP
    php -S localhost:8000 -t public
